| ------------ | ----------------------------------------------------------------------------- |
| `number`     | Representa tanto enteros como flotantes. Ej: `42`, `3.14`                     |
| `string`     | Representa cadenas de texto. Ej: `"hello"`, `'world'`                         |

| `null`       | Representa un valor nulo.                                                     |
| `undefined`  | Representa una variable que no ha sido definida.                              |
