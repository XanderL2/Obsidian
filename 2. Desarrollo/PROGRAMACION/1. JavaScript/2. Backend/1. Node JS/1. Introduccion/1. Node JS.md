## Que es Node JS: 
- Node JS es un **entorno de ejecucion** de JS que permite correr JavaScript a nivel de sistema.
- Permite usar **CommonJS** y **ESModules**


### Instalacion:
- **Verficiar version**
	```sh
	node -v
	```

- **Interpretaar Node a traves de Lineaa de comandos**
	```sh
	node
	```
