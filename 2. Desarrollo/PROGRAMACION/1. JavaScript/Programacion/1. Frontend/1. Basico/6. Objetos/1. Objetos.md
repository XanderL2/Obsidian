- **Uso del This**
	- *El this solo se puede usar en funciones que usan function, no en funciones flecha*.

	```javascript

	let person = {
	  'name': 'pipe',
	  'age': 20,
	  'greet': () => console.log(this.name) // âŒ La funcion flecha no puede usar this
	}
	person.greet();


	// âœ… Las funciones normales si pueden usar this
	let person = {
	  'name': 'pipe',
	  'age': 20,
	  'greet': function() {
		  console.log(this.name)
	  }
	}
	person.greet();
	```
