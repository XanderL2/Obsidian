## Que son los CallBacks?
- Una funcion que se ejecutara cuando un proceso asincrono termine.
- Son muy parecidos a los `{javascript icon} AddEventListener()`, en los cuales dejas codigo ahi, sin embargo esta se ejecutara cuando el resultado se obtenga.
- Suelen presentar 2 parametros `{javascript icon} callback(err, data)`

### Ejemplo de CallBack:

```js
function OperacionAsincrona(callback) { // ðŸš§ La operacion asincrona pide el Callback

    console.log("Cargando datos..")

    // ðŸš§ Simulando una operacion asincrona - Cuando pasen 2 segundos se ejecutara el codigo
    setTimeout(() => {

      const error = null;
      const data = "data obtenida"
  

      // âœ… Cuando la opercion asincrona termine ejecutaremos el Callback
      callback(error, data);      

    }, 2000);
  
}


// ðŸ‘‡ Llamada al proceso asincrono pasandole el Callback

  }else {
    console.log(data)
  }

});

```


#### Porque no se debe usar un Callback:
- Es mucho mas dificil de leer y entender.
- Cuando se quieren anidar operaciones asincronas se vuelve mas dificil y aparece el conocido `{js icon} CallbackHell ðŸ’€`.
