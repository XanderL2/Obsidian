## Eventos de Mouse:

| **Eventos de Clickl**                                 | **Accion:**                                                                       |
| ----------------------------------------------------- | --------------------------------------------------------------------------------- |
| `{javascript icon title:clickEvent} click`            | Se ejecuta cuando se realiza un click sobre el elemento.                          |
| `{javascript icon title:ContextMenu} contextmenu`<br> | Se ejecuta cuando se hace un click derecho o anticlick en el elemento.            |
| `{javascript icon title:DobleClick} dblclick`         | Se ejecuta cuando se hace doble click en un elemento.                             |
| `{javascript icon title:MouseDown} mousedown`         | Se ejecuta cuando se presiona el click.                                           |
| `{javascript icon title:MouseUp} mouseUp`             | Se ejecuta cuando el click ha sido levantado.                                     |
| **Eventos de movimiento del Mouse Compartidos**       | **Este tipo de eventos incluyen a sus hijos tambien.**                            |
| `{javascript icon title:MouseOver} mouseover`         | Se ejecuta cuando se pasa el mouse en el elemento o en cualquiera de sus hijos.   |
| `{javascript icon title:MouseOut} mouseout`           | Se ejecuta cuando se pasa el mouse sale  del elemento de cualquiera de sus hijos. |
| **Eventos de movimiento del Mouse Individuales**      | **Esto significa que el evento solo sera agregado para ese elemento en concreto** |
| `{javascript icon title:MouseEnter} mouseenter`<br>   | Se ejecuta cuando el mouse se pasa por el elemento de forma individual.           |
| `{javascript icon title:MouseMove} mouseleave`        | Se ejecuta cuando el mouse sale del elemento de forma individual.                 |
