### Uso de Angular CLI:
- **Crear un Nuevo Proyecto:**
	```sh
	ng new nombreProyecto
	```
	
- **Compilar Codigo a JS (Solo para despliegue):**
	```sh
	ng build
	```
- **Generar Modulos, Componentes, Servicios, etc:**
	```sh
	ng generate
	```
- **Arrancar proyecto:**
	```sh
	ng serve
	```

