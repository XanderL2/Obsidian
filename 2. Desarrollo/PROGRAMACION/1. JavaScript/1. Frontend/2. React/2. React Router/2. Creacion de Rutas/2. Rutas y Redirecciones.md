## Para crear Redirecciones:
- Para crear redirecciones usaremos el componentes **Link** de React Router.
- Este nos servira para redireccionar al usuario a una pagina en concreto
### Para crear Redirecciones:

```tsx
// Debemos importar el componente      ðŸ‘‡    para redireccionar
import { BrowserRouter, Routes, Route, Link } from "react-router-dom"; 
import { DashBoard, Landing, Root } from "./pages/index";

function App() {
  return (
    <BrowserRouter>
      <Navigation /> ðŸ‘ˆ Este componente aparecera en todas las pagiinas

      <Routes>
        <Route path="/" element={<Root />} />
        <Route path="/landing" element={<Landing />} />
        <Route path="/dashboard" element={<DashBoard />} />
      </Routes>

    </BrowserRouter>
  );
}


ðŸ‘‡  Para redireccionar
function Navigation() {
  return (
    <nav>
      <ul>
        <li>
          <Link to="/landing">Esto llevara a landing</Link> ðŸ‘ˆ Asi se usa la etiqueta
        </li>

        <li>
          <Link to="/dashboard">Esto llevara a dashboard</Link>
        </li>
      </ul>
    </nav>
  );
}

export default App;
```
