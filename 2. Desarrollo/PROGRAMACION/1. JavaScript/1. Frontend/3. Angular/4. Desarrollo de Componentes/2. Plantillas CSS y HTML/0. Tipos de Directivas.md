### Tipos de directivas:

| **Tipo Directiva**    | Representacion                                             |
| --------------------- | ---------------------------------------------------------- |
| `{html icon} [style]` | Los Corchetes representan directivas que son de atributos. |
| `{html icon} (click)` | Los parentesis representan directivas que son de eventos   |
