## ¿Que es la Screaming Architecture?
Es una forma de Organizacion de Proyectos que se basa en:

- **Organizacion por Funcionalidades**
- **Mantiene las Cosas Separadas**
- **Usa Interfaces**

## Ejemplo:


```
mi-aplicacion/
├── Tareas/                      <- Todo lo relacionado con las tareas
│   ├── Entidades/               <- Clases que representan las tareas
│   │   └── Tarea.py
│   ├── CasosDeUso/              <- Funcionalidades específicas de las tareas
│   │   ├── CrearTarea.py        <- Creación de nuevas tareas
│   │   ├── EditarTarea.py       <- Edición de tareas existentes
│   │   ├── BorrarTarea.py       <- Eliminación de tareas
│   │   └── ListarTareas.py      <- Listado de todas las tareas
│   ├── Interfaces/              <- Interfaces para interactuar con las tareas
│   │   └── RepositorioDeTareas.py
│   └── Repositorios/            <- Implementaciones de almacenamiento de tareas
│       └── RepositorioEnMemoria.py

├── Listas/                      <- Todo lo relacionado con las listas
│   ├── Entidades/               <- Clases que representan las listas
│   │   └── Lista.py
│   ├── CasosDeUso/              <- Funcionalidades específicas de las listas
│   │   ├── CrearLista.py        <- Creación de nuevas listas
│   │   ├── EditarLista.py       <- Edición de listas existentes
│   │   ├── BorrarLista.py       <- Eliminación de listas
│   │   └── ListarListas.py      <- Listado de todas las listas
│   ├── Interfaces/              <- Interfaces para interactuar con las listas
│   │   └── RepositorioDeListas.py
│   └── Repositorios/            <- Implementaciones de almacenamiento de listas
│       └── RepositorioEnMemoria.py

├── Adaptadores/                 <- Conectores con el mundo exterior
│   ├── Controladores/           <- Manejan las interacciones del usuario
│   │   ├── TareaController.py
│   │   └── ListaController.py
│   ├── Gateways/                <- Conectan con bases de datos u otros servicios
│   │   ├── RepositorioSQLDeTareas.py
│   │   └── RepositorioSQLDeListas.py
│   └── Presentadores/           <- Preparan los datos para mostrar en la interfaz
│       ├── PresentadorDeTareas.py
│       └── PresentadorDeListas.py

├── Infraestructura/             <- Configuración técnica del proyecto
│   ├── BaseDeDatos/             <- Configuración de la base de datos
│   │   └── ConfiguracionDeDB.py
│   ├── Web/                     <- Configuración del servidor web
│   │   └── Servidor.py
│   └── OtrosServicios/          <- Configuraciones para otros servicios externos
│       └── ConfiguracionDeOtrosServicios.py

├── Config/                      <- Archivos de configuración del proyecto
│   └── configuracion.yml

├── Pruebas/                     <- Pruebas para asegurar el funcionamiento correcto
│   ├── Unitarias/               <- Pruebas individuales de cada parte del código
│   │   ├── PruebaCrearTarea.py
│   │   ├── PruebaEditarTarea.py
│   │   ├── PruebaCrearLista.py
│   │   └── PruebaEditarLista.py
│   └── Integracion/             <- Pruebas que verifican la integración entre epartes
│       ├── PruebaRepositorioDeTareas.py
│       └── PruebaRepositorioDeListas.py

├── Scripts/                     <- Scripts auxiliares para el proyecto
│   └── ejecutar_servidor.sh

├── .gitignore                   <- Archivos y carpetas a ignorar en Git
├── README.md                    <- Documentación del proyecto
└── LICENSE                      <- Información sobre la licencia del proyecto

```
