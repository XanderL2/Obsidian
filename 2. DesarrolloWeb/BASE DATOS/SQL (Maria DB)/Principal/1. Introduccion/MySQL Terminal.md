## Comandos SQL Disponibles:

- **Ver Bases de Datos Disponibles
	```sql
	SHOW DATABASES;
	```
	
-  **Usar una Base de Datos:**
	```sql
	USE nombreDatabase;
	```
	
- **Mostrar Tablas de una BD:**
	```sql
	SHOW TABLES;
	```
- **Ver la estructura de una Tabla:**
	```sql
	DESCRRBE nombreTabla;
	```
- **Dejar de usar una BD:**
	```sql
	SHOW DATABASES;
	```
