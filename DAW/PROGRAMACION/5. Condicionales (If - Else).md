## **Condicional:**

Se usa para condicionar el codigo y controlar el flujo del programa, segun se cumpla o no una condicion.

## **Estructura Basica de un Condicional:**

- if = Si
- else if = Caso contrario si.
- else = Caso contrario

```
if(condicion){
	//Pasos segun la condicion
}else if(conficion){
	//Pasos segun la condicion
}else{
	//Pasos segun la condicion
}
```
